# Autism Client PvP

A autism client addon that focuses on PvP and suggestions from the discord server.

## This is an addon and requires Autism Client

The original repo for Autism Client can be found [here](https://github.com/AutismDevelopment/Autism-Client/tree/main)

# What's Included?

## Modules:

- **AimAssist+**
- **AntiHunger**
- **Auto Anchor**
- **Auto Crit**
- **Auto Crystal**
- **Auto Totem**
- **BetterBoatfly**
- **FastPlace**
- **Hover Totem**
- **MaceSwap**
- **PearlCatch**
- **Seed X-Ray** (Uses highly-accurate native Cubiomes simulation, works fully standalone, executes asynchronously on background thread pools, and automatically retrieves integrated server seeds in singleplayer)
- **SnapTap**
- **WTap**
